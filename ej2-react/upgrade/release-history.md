# Release History

Each new release of Syncfusion<sup style="font-size:70%">&reg;</sup> delivers features, fixes, and improvements. Refer to the Syncfusion<sup style="font-size:70%">&reg;</sup> React [release notes](https://ej2.syncfusion.com/react/documentation/release-notes/index/) for detailed information about changes in each release.

For guidance on upgrading, consult the React [Upgrade Guide](https://help.syncfusion.com/upgrade-guide/react-ui-components). The guide covers the following aspects that are critical for a successful upgrade:

* **Breaking changes:** Modifications that may impact existing code and require updates.
* **Bug fixes:** Resolved issues that improve component stability and reliability.
* **Features:** New functionality and enhancements introduced in each release.
* **Known issues:** Identified issues to consider between the current and target versions during upgrade planning.
