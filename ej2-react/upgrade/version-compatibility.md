# Syncfusion<sup style="font-size:70%">&reg;</sup> React supported versions

## React version compatibility

The following table lists React versions and the minimum supported Syncfusion<sup style="font-size:70%">&reg;</sup> React package versions.

| React version | Minimum supported Syncfusion React package version |
| ------------- | ----------------------------------------------- |
| [React v19](https://reactjs.org/) | 29.1.33 and above |
| [React v18](https://reactjs.org/blog/2022/03/29/react-v18.html) | 20.2.36 and above |
| [React v17](https://reactjs.org/blog/2020/10/20/react-v17.html) | 18.3.50 and above |
| [React v16](https://reactjs.org/blog/2017/09/26/react-v16.0.html) | 16.2.45 and above |

## Syncfusion<sup style="font-size:70%">&reg;</sup> version information

Syncfusion<sup style="font-size:70%">&reg;</sup> follows a quarterly release schedule, publishing a major volume every three months. Releases use a **Major.Minor.Revision** versioning scheme to make change tracking predictable:

For example, version `22.1.34` is interpreted as:

* **22** — major volume (quarterly release). Major volumes may include significant updates and possible breaking changes.
* **1** — minor release within the major volume (feature additions and bug fixes that are generally non-breaking).
* **34** — revision or patch number (bug fixes and small updates).

Refer to the release notes and the upgrade guide to verify compatibility and identify any breaking changes before updating packages.
